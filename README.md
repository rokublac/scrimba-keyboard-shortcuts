## Windows

#### General

| Command | Action |
| --- | --- |
| `Ctrl + S` | Execute code |
| `Ctrl + B` | Toggle mini browser focus |
| `Ctrl + U` | Show/hide mini browser |
| `Ctrl + Y` | Show/hide console|
